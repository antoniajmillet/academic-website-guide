# Sources, Inspiration, and Code Provenance

This file records the websites, documentation, and standard web features that informed the guide and reference site.

## Important distinction

The reference website was not copied from any single academic website. Several sites were reviewed to compare common ways of organizing an academic identity, displaying projects, linking a CV, and using photographs. The HTML, CSS, and JavaScript in reference-website/ were developed for Antonia J. Millet’s website and reorganized for this educational guide.

A publicly viewable website was treated as design inspiration unless its repository and reuse license clearly permitted code reuse.

## Feature-level provenance

| Feature | How it was developed |
| --- | --- |
| One-page academic structure | Common organizational pattern adapted for this project |
| Sticky navigation | Custom HTML and CSS using standard web positioning |
| Expandable research cards | Custom use of native `<details>` and `<summary>` elements |
| Color variables | Custom CSS properties |
| Desktop and mobile layouts | Custom CSS media queries |
| Current footer year | Small JavaScript function using the current date |
| QR redirect | Custom redirect page using JavaScript, meta refresh, a fallback link, and UTM parameters |
| Publishing and indexing instructions | Summarized from official GitHub and Google documentation |
