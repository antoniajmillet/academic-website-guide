# Example Academic Website

This folder contains the complete working website used throughout the guide.

Live example:

https://antoniajmillet.github.io/

## Main files

- `index.html` — visible content and structure
- `styles.css` — colors, typography, spacing, cards, and responsive layout
- `script.js` — small automatic and mobile behaviors
- `robots.txt` — crawler instructions
- `sitemap.xml` — main page information for search engines
- `assets/images/` — website photographs
- `assets/documents/` — CV PDF
- `go/website/` — permanent QR redirect and QR files

## Open locally

You can double-click `index.html`, or open this folder with the Live Server extension in Visual Studio Code.
